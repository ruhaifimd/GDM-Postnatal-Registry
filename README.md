# GDM-Postnatal-Registry
**GDM Postnatal e-Registry** adalah sebuah aplikasi web berpusat yang direka untuk membantu kakitangan kesihatan memantau dan merekod data postnatal pesakit yang pernah didiagnosis dengan *Gestational Diabetes Mellitus* (GDM) semasa mengandung.

Sistem ini memastikan setiap pesakit GDM mendapat pemantauan berterusan selepas bersalin — bermula dari ujian OGTT 6 minggu postnatal hinggalah kepada ujian darah tahunan sehingga tahun 2030 — selaras dengan garis panduan **CPG Pengurusan Diabetes Mellitus Jenis 2, Edisi ke-6, Kementerian Kesihatan Malaysia (2020)**.
